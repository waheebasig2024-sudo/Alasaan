## نظرة عامة

تطبيق مساعد شخصي ذكي للأندرويد بالكامل، مبني بـ **React Native + Expo + TypeScript + Expo Router**.
يعمل بصوت عربي طبيعي ويفهم اللهجة اليمنية وسائر اللهجات العربية.

---

## مسار القرار (إجباري)

```
1. Tools First   → تنفيذ الأوامر مباشرةً (كاميرا، اتصال، خرائط...)
2. Memory Second → استدعاء المعلومات الشخصية المحفوظة
3. Gemini Third  → الإجابة على الأسئلة عبر Gemini AI
4. No Hallucination → لا اختلاق — رد صادق عند الفشل
```

---

## الميزات الأساسية

- **الأوامر الصوتية**: اضغط زر الميكروفون وتحدث — يتوقف تلقائياً عند الصمت وينفذ الأمر
- **كلمة التنبيه**: قل "يا الحسن" أو "يا حسن" وسيستيقظ مباشرةً
- **الصوت الناطق**: يرد بصوت عربي طبيعي بعد كل أمر أو سؤال (expo-speech)
- **العمل في الخلفية**: المايكروفون يعمل حتى عندما يكون التطبيق في الخلفية
- **الذاكرة الشخصية**: يتذكر اسمك وتفضيلاتك وأشخاصك المهمين
- **اللهجة اليمنية**: يفهم اللهجة اليمنية ويرد بأسلوب عربي طبيعي
- **دوران المفاتيح الذكي**: ثلاثة مفاتيح Gemini تدور تلقائياً عند انتهاء حصة أي منها
- **Android Native-first**: لا WebView، لا localhost، لا Termux

## التقنيات المستخدمة

- **React Native + Expo** (~54)
- **TypeScript** (~5.9)
- **Expo Router** (~6)
- **Expo Speech** — صوت ناطق بالعربية
- **Expo AV** — تسجيل صوتي مع كشف الصمت التلقائي
- **Zustand** — إدارة الحالة
- **Google Gemini AI** (gemini-2.5-flash) عبر خادم Node.js/Express

---

## متطلبات الأندرويد

- **الحد الأدنى**: Android 9 / API 28
- **مستهدف**: Android 14 / API 34
- **البنية**: APK (تطوير) / AAB (إنتاج)

## البناء

```bash
# تثبيت الحزم
pnpm install

# بناء APK للاختبار
eas build --profile android --platform android

# بناء AAB للنشر
eas build --profile production --platform android
```

## الصلاحيات المطلوبة

- `RECORD_AUDIO` — تسجيل الصوت وكلمة التنبيه
- `FOREGROUND_SERVICE_MICROPHONE` — المايكروفون في الخلفية
- `CALL_PHONE` — الاتصال بجهات الاتصال
- `READ_CONTACTS / WRITE_CONTACTS` — الوصول لجهات الاتصال
- `ACCESS_FINE_LOCATION` — تحديد الموقع
- `CAMERA` — الكاميرا
- `READ_CALENDAR / WRITE_CALENDAR` — التقويم
- `VIBRATE / WAKE_LOCK` — الاهتزاز والإبقاء نشطاً
