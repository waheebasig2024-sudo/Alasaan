# Hassan Voice Assets — أصوات الحسن

هذا المجلد مخصص لأصوات الحسن المُعالَجة محلياً.

## الهيكل

```
hassan_voice/
├── responses/          # ردود صوتية جاهزة (mp3)
│   ├── greeting.mp3    # "مرحباً، أنا الحسن"
│   ├── thinking.mp3    # صوت التفكير
│   ├── done.mp3        # "تم بنجاح"
│   ├── error.mp3       # "حدث خطأ"
│   └── alert.mp3       # تنبيه
├── tones/              # نغمات قصيرة للتغذية الراجعة
│   ├── ping.mp3
│   ├── success.mp3
│   └── warning.mp3
└── README.md
```

## مصادر الصوت (من الأعلى للأدنى أولوية)

1. **Aiden Server TTS** — `POST /api/tts` — الصوت العربي الاحترافي من خادم Aiden
2. **Gemini 2.5 Flash TTS** — صوت Orus العربي مباشرة من Google
3. **الأصوات المحلية** — ملفات mp3 في هذا المجلد (للردود المتكررة)
4. **expo-speech** — احتياط طارئ فقط (معطَّل افتراضياً)

## إضافة أصوات جديدة

ضع ملفات `.mp3` أو `.wav` في المجلد المناسب وارجع إليها في:
`src/services/hassan-tts.service.ts`

## متطلبات جودة الصوت

- صيغة: MP3 أو WAV
- معدل أخذ العينات: 22050 Hz أو 44100 Hz
- القنوات: Mono أو Stereo
- البت ريت: 128 kbps أو أعلى
- اللغة: عربي فصيح (ar-SA) بنبرة تقنية احترافية
